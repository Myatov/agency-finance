# Счета, отчёты и закрывающие документы — где вносить и как смотреть

## Где вносить (одна точка входа)

**Карточка периода** — единственное место, где вносятся счета, отчёт по периоду и оплаты:

1. **Оплаты** → в таблице у строки с уже созданным периодом нажмите «Периоды» → откроется список периодов услуги → у нужного периода «Счета и оплаты».
2. Либо **Услуги** → выберите услугу → **Периоды** → у нужного периода «Счета и оплаты».

В карточке периода (`/periods/[id]`) доступны:
- добавление **счёта** (сумма, юрлицо);
- прикрепление **отчёта по периоду** (файл, тип: предоплата/постоплата/дробная);
- по каждому счёту — **оплаты**.

**Закрывающие документы** ведутся отдельно: раздел меню **Закрывающие документы** (пакеты по клиентам, загрузка документов).

## Как смотреть в разрезе Клиента, Сайта, Услуги

Сейчас:
- **Оплаты** — свод по всем периодам (с фильтром по клиенту и аккаунту), без перехода в карточки клиента/сайта.
- **Услуга → Периоды** — все периоды одной услуги и переход в карточку периода (счета, отчёт, оплаты).

Предлагаемое развитие (при необходимости):

| Уровень | Что показывать | Откуда вносить |
|--------|-----------------|----------------|
| **Клиент** | Страница клиента: список сайтов и по каждому — услуги и периоды с кратким статусом (есть счёт/отчёт/оплата), ссылки на карточку периода | Внесение по-прежнему только в карточке периода; со страницы клиента — только переход «Счета и оплаты» по периоду |
| **Сайт** | Страница сайта: список услуг и по каждой — периоды с тем же статусом и ссылками на карточку периода | Аналогично — только просмотр и переход в период |
| **Услуга** | Уже есть: Услуги → [услуга] → Периоды → по каждому периоду «Счета и оплаты» | Внесение в карточке периода |

Итого: **вносить** — только в карточке периода; **смотреть** — в отчёте «Оплаты» (свод + фильтры) и при необходимости на будущих страницах клиента/сайта как агрегация со ссылками на периоды.
